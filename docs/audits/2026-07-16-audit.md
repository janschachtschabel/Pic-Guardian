# Code-Audit — Bild-Lizenz-Check

**Datum:** 2026-07-16 · **Methode:** Whole-Codebase, multi-role (Security /
Correctness / Frontend), evidenzbasiert · **Scanner:** ruff, npm audit, madge

## 1. Executive Summary

Der Bild-Lizenz-Prüfdienst (FastAPI + Angular 21) ist funktional solide und gut
strukturiert. Der Audit fand **eine kritische Sicherheitslücke (SSRF)**, mehrere
**systematische Correctness-Bugs** in den Heuristiken (Substring-Fehltreffer,
Domain-Spoofing, falsches Lizenz-Label) und einen **Prod-Blocker im Frontend**
(hardcodierte Backend-URL). **Alle diese Punkte wurden im Zuge des Audits
behoben und mit Regressionstests abgesichert.** Wichtigste verbleibende Aufgabe:
Der Dienst hat **keine App-interne Authentifizierung** — er muss hinter einem
Auth-/Reverse-Proxy und über HTTPS betrieben werden.

## 2. Overview

- **Stack:** Backend FastAPI/Python 3.12 (~3.000 LOC, 19 Module), Frontend
  Angular 21 + Material 3 (~1.150 LOC). Persistenz: JSON-Risikospeicher +
  In-Memory-Batch-Jobs (keine DB).
- **Zweck:** Erkennung problematischer Bildlizenzen vor der Vorschau-Auslieferung
  (Default-Deny), 11 freie Prüfschritte, Batch über Sammlungen/CSV.
- **Abdeckung:** Alle Backend-Module (Loader, edu-sharing-Client, Checks,
  Aggregation, Batch, main) und alle Frontend-Dateien line-by-line gelesen.
  Scanner ohne Befund (0 CVEs, keine Zyklen).

## 4. Verdict

**Conditional production-ready.** Die funktionale und die Kern-Sicherheitsqualität
sind gegeben. Bedingungen:
1. **Hinter Authentifizierung + Reverse-Proxy** betreiben (kein App-internes
   Auth-Modell) — sonst sind `/api/check`, `/api/batch/*` und `/api/risk-hub`
   offen erreichbar.
2. **HTTPS erzwingen** (Auth-Header, Bildübertragung).
3. Optional: Rate-Limiting für die ressourcenintensiven Endpoints.

## 5. Behobene Findings (mit Beleg)

### 🔴 Kritisch
- **SSRF** (`image_source.py`, `edu_sharing.py`, `c10`): nutzergesteuerte URLs
  (Bild-URL, Fundseite/`ccm:wwwurl`, robots.txt) erreichten interne Ziele
  (127.0.0.1, 169.254.169.254 Cloud-Metadata). **Fix:** neues `net_guard.py` —
  Host-Auflösung + Block privater/loopback/link-local/reserved-IPs, manuelles
  Redirect-Handling mit Re-Validierung pro Hop, Content-Length-Vorabprüfung. In
  alle nutzergesteuerten Fetch-Sinks eingebunden. Verifiziert über API (422).
- **Frontend hardcodierte URL** `http://localhost:8000/api` (`api.service.ts`):
  Prod funktionslos + Mixed-Content + Auth im Klartext. **Fix:** relativer Pfad
  `/api` + `proxy.conf.json` (Dev) / Reverse-Proxy (Prod). Verifiziert (GET+POST
  über `:4200/api`).

### 🟠 Hoch (Correctness / Robustheit)
- **`contains_agency`/`contains_free_source` Substring-Fehltreffer** (`wordlists.py`):
  „dpa" in „Sandpapier", „nasa" in „nasal" → systematisch falsches RED/GREEN über
  6 Checks. **Fix:** wortgrenzen-gebundene Regex-Alternation.
- **Domain-Spoofing** (`c02` `_host_matches`): `"pexels.com" in "pexels.com.evil.ru"`
  → falsches GREEN/RED. **Fix:** nur exakter Host oder echte Subdomain.
- **Falsches Lizenz-Label** (`wordlists.py` `find_cc_license`): „CC BY SA 4.0"
  statt „CC BY-SA 4.0" + No-op-`replace`; erfundene Version 4.0. **Fix:**
  Bindestrich-Label, versionslose Deed-URI ohne erkannte Version.
- **Size-/Decompression-Limits** (`image_source.py`): Byte-Limit erst nach
  Vollpuffer; kein Pixel-Limit. **Fix:** Content-Length-Vorabprüfung +
  `Image.MAX_IMAGE_PIXELS = 64 MP`.
- **Pfad-Injection** (`edu_sharing.py`): `node_id`/`coll_id` roh in URL-Pfad.
  **Fix:** `quote(..., safe="")`.
- **Batch: Poll-Fehler sperrt Start-Button** (`batch.component.ts`): **Fix:**
  Job-Status im Error-Handler auf `error`.
- **`submit()` ohne In-Flight-Sperre** (`app.ts`): Enter-Spam → parallele
  Requests. **Fix:** `if (loading()) return`.

### 🟡 Mittel
- **c03 WebStatement** pauschal RED → nur `plus:LicensorURL`/Agentur-Domain RED.
- **JobStore-Thread-Safety** (`batch.py`): Mutationen/Snapshots jetzt unter Lock.
- **CSV-Auth an alle Repos** (`main.py`): Credentials nur ans Default-Repo; CSV
  auf 5000 Zeilen begrenzt.
- **Risikospeicher fail-open** (`risk_hub.py`): Log statt stiller Leerung +
  atomares Speichern (`os.replace`).
- **`normalize_license_uri`** globales Replace/Query-Korruption → `urlsplit`.
- **c10 robots.txt** ungenauer Substring-Test → spezifische TDM-Marker + Guard.
- **A11y** (`app.html`, `batch.component.html`): `role="alert"` an Fehlern,
  `aria-label` an Count-Chips.
- **Zahlenfeld-Coercion** (`batch.component.ts`): leeres Feld → Default statt `"null"`.
- **Pagination** (`edu_sharing.py`): harte Iterationsobergrenze.

### 🟢 Niedrig
- Toter Code (`aggregate.py`), Lint (ungenutzter Import, f-string, Importreihenfolge,
  ungenutzter `DecimalPipe`) — alle behoben, ruff/ng sauber.
